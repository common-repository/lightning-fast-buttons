Lightning-Fast Buttons is a WordPress plugin will help you to keep your website lean with minimum number of requests and additional file size.

== Description ==

Lightning-Fast Buttons plugin is a lightweight and fast loading alternative to typical social media sharing plugins which add hundreds of kilobytes and dozens of HTTP requests to your website. This plugin uses editable SVG icons to minimize number of requests and optionally queries Sharedcount.com API for number of shares.

= Features =
* 8 buttons for the most popular social network (Twitter, Facebook, LinkedIn, Google+, Reddit, Tumblr, StumbleUpon, Pinterest)  
* Share count for Facebook, LinkedIn, Google+, Pinterest and StumbleUpon
* Simple, clean and responsive default design which can be easily updated to your liking
* Only 3 requests and 3.5kB of code
* Vanilla JS to query Sharedcount.com API, no jQuery dependency
* No cookies or tracking code
* Position above or below the post
* Use [lfbtn] shortcode to place the buttons manually. The shortcode has no extra options
* Supports Polish and English languages

== Installation  ==
* Install the plugin either via the WordPress.org plugin directory, or by uploading the files to your server (in the /wp-content/plugins/ directory).
* Activate the Lightning-Fast Buttons plugin through the 'Plugins' menu in WordPress.
* Go to Lightning-Fast Buttons in sidebar and display or hide selected social media buttons.
* Register at SharedCount to get your free API key to enable share count for selected buttons.
* If you want to override default SVG icons paste their parameters to the respective fields.
